📥 Import Bins from CSV

Paste your CSV below or upload a file. Each row = one bin.

Required column: num (bin number/name)
Optional columns: type, size, color, damage, status, notes
Example: num,size,color,status
14YD-GREEN-1,14 yard,green,in
7YD-BLACK-2,7 yard,black,out
📜 Import Bin History

Import legacy bin rental history. Each row = one historical rental.

Required: bin_num
Optional: bin_size, customer_name, customer_phone, job_address, city, dropoff_date, pickup_date, material_type, notes
Example: bin_num,bin_size,customer_name,dropoff_date,pickup_date
14YD-GREEN-1,14 yard,John Smith,2025-01-15,2025-01-22
